# bilibili-cli-study

> 深入研究 [bilibili-cli](https://github.com/public-clis/bilibili-cli) 的技术实现

## 项目简介

bilibili-cli 是一个 Bilibili 命令行工具，本项目通过源码阅读和实验，深入理解其架构设计与核心实现。

## 目录结构

```
bilibili-cli-study/
├── bilibili-cli/    # bilibili-cli 源码
├── notes/           # 学习笔记
└── scripts/         # 实验脚本
```

## 学习内容

- CLI 工具架构设计
- Bilibili API 交互方式
- 命令行交互模式实现
